**Professional Remote Desktop Connection Manager for Linux**  
Created by John Avendaño

## ✨ Features

- ✅ **Multiple Connections** - Manage all your RDP connections in one place
- ✅ **Professional UI** - Beautiful terminal interface with colors
- ✅ **Secure** - Passwords never stored, asked each time
- ✅ **Backup & Restore** - Never lose your configurations
- ✅ **Export/Share** - Share connection settings with your team
- ✅ **Logs** - Automatic logging of all connections
- ✅ **Desktop Shortcuts** - One-click connections from your desktop
- ✅ **Custom Icons** - Personalize your shortcuts
- ✅ **Audio & Drive Redirection** - Full remote desktop experience

## 🖥️ Compatible With

- Zorin OS 16/17
- Ubuntu 20.04/22.04/24.04
- Linux Mint 20/21
- Debian 11/12
- Any Debian-based distribution

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/johnavendano-afk/rdp-manager
cd rdp-manager

# Make the script executable
chmod +x rdp-manager-pro.sh

# Run the manager
./rdp-manager-pro.sh
